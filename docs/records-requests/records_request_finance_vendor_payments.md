### Request

Pursuant to Iowa Code § 22.1 et seq., I request the following public records:

#### 1. FY27 Vendor Payment Report for Account 522101 (Computer Software)

Please provide a list or report showing all vendor payments made from **Account 522101 — Computer Software** during Fiscal Year 2027 (July 1, 2026 – June 30, 2027).

Specifically, I request the following fields for each payment:
- Vendor name
- Payment amount
- Payment date or date range
- Invoice number or reference (if available)
- Description or purpose (if available in the system)

#### 2. FY27 Vendor Payment Report for Account 522102 (IT Services — External)

Please provide the same vendor payment information for **Account 522102 — IT Services — External** for FY27.

#### 3. FY27 Vendor Payment Report for Account 521108 (Other Professional Services)

Please provide the same vendor payment information for **Account 521108 — Other Professional Services** for FY27.

| Account | FY26 Adopted | FY27 Requested | Increase |
